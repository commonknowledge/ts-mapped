The Movement Data Library (MDL) is a curated collection of datasets made available by Mapped for use in your maps. Instead of sourcing and uploading your own geographic data, you can add MDL datasets directly — useful for contextual layers like demographic data, deprivation indices, or other publicly available information.

## Browsing available datasets

### 1. Go to Data Sources

Navigate to **Data Sources** from the main navigation.

### 2. Open the Movement Data Library tab

Click the **Movement Data Library** tab. You'll see a list of available datasets organised by category.

### 3. Browse and preview

Click any dataset to see a description, the columns it contains, and a preview of the data. This helps you decide whether it's relevant before adding it to a map.

## Adding an MDL dataset to your organisation

### 1. Select a dataset

From the Movement Data Library tab, click the dataset you want to use.

### 2. Click "Add to organisation"

Click **Add to organisation**. The dataset is now available as a data source within your organisation, just like any data source you've connected yourself.

You don't need to manage credentials or syncing — MDL datasets are maintained by Mapped and updated automatically.
