Action Network is a popular organising platform. Connecting it to Mapped lets you visualise your supporters, signers, and attendees geographically.

## Before you start

You'll need an Action Network account and your API key. You can find this in Action Network under **Developer → API & Sync**.

## Connecting your account

### 1. Create a new data source

Go to **Data Sources**, click **New data source**, and select **Action Network**.

### 2. Enter your API key

Paste your Action Network API key into the field provided.

### 3. Select a resource

Choose what you want to import — people, event attendees, petition signers, or another supported resource type.

### 4. Map your location fields

Action Network stores addresses in structured fields. Select the appropriate fields for street address and postcode so Mapped can geocode your records.

### 5. Save and sync

Name your data source and click **Save**. Mapped will perform an initial import and will re-sync automatically on a regular schedule to pick up new and updated records.
