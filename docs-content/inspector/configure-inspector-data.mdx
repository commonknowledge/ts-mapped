When you click a marker on the map, the Inspector's Data tab shows field values from your connected data sources. If you have multiple data sources connected, you can control which ones appear and in what order.

## Opening the data configuration

### 1. Click a marker

Click any marker on the map to open the Inspector.

### 2. Open the Data tab

Select the **Data** tab at the top of the Inspector panel.

### 3. Click the settings icon

Click the cog (⚙) icon within the Data tab. The Inspector Data Config panel opens, listing all connected data sources.

## Configuring which sources appear

- **Toggle a data source on or off** — click the toggle next to a data source name to show or hide it in the Data tab. Hiding a source removes it from the Inspector but doesn't affect the data itself.
- **Reorder data sources** — drag a data source up or down to change the order they appear in. Put the most relevant source at the top.

Click outside the config panel to close it. Your preferences are saved for that map.

## When this is useful

If you've connected several data sources (e.g. a members list, an events attendance list, and a canvassing responses list), the Inspector could become cluttered. Hiding less relevant sources and putting the primary one at the top keeps the view focused.
